Siteplan for Kelowna Longboard Alliance

Site Name

The site name will be "Kelowna Longboard Alliance". This is the same name as the club. If I could pick a domain it would be kla.ca

Site Purpose

This site will provide information on the local longboard club, including what kinds of longboarding we do. It will include some longboard media and links to media, a way to get in contact with members of the club, and a list of local longboard events.

Scenarios

What is downhill longboarding? How/where can I learn to longboard/get better at longboarding? How can I meet up with the community?

Typography

This page will use the Roboto font for everything. While I want this page to look nice, I don't need it to look gorgeous. It's main purpose is to convey information, so I chose one font for simplicities sake.
